O que é o KernelSU?

KernelSU é uma solução de root baseada no kernel para dispositivos Android. Ele funciona diretamente no kernel Linux, oferecendo mais controle sobre aplicativos do espaço do usuário.

Características principais do KernelSU:

  • Controle de Acesso Root: Apenas aplicativos autorizados podem acessar ou ver o comando su.
  • Privilégios Root Personalizáveis: Permite a personalização de su com uid, gid, grupos, capacidades e regras do SELinux.
  • Módulos: Permite modificar o sistema sem sistema usando OverlayFS, oferecendo grande poder.

Diferenças entre Root Normal e KernelSU:

Root Normal:

Tradicionalmente, o root no Android é obtido modificando o ramdisk ou instalando um gerenciador de root como o Magisk. Isso permite que aplicativos do usuário obtenham privilégios de root, mas o processo de root é visível para o sistema e pode ser detectado por algumas aplicações.

KernelSU:

Funciona diretamente no kernel Linux, oferecendo uma camada adicional de privilégios. Com o KernelSU, apenas aplicativos autorizados podem acessar o comando su, e o processo de root é menos visível ao sistema, tornando-o mais seguro e menos detectável por aplicativos.

Em resumo, o KernelSU oferece uma abordagem mais avançada e segura para obter privilégios de root, enquanto o root normal é mais tradicional e visível ao sistema.
