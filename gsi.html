O que são GSIs?

GSIs (Generic System Images) são imagens de sistema Android baseadas no código-fonte do AOSP (Android Open Source Project) sem modificações.

Principais características das GSIs:

  • Compatibilidade: Podem ser instaladas em dispositivos compatíveis com o Project Treble (a partir do Android 9) e que suportam o Vendor Native Development Kit (VNDK).
  • Desenvolvimento e Testes: Utilizadas principalmente por desenvolvedores para testar aplicativos em diferentes dispositivos e versões do Android.
  • Sem Personalizações: Não incluem modificações de fabricantes, oferecendo uma experiência pura de Android.
  • Atualizações e Versões: Disponíveis para diversas versões do Android, incluindo betas e versões de desenvolvimento.

O uso de GSIs é indicado para dispositivos descontinuados ou que não receberão mais atualizações do fabricante. No entanto, como são genéricas, podem não funcionar perfeitamente em todos os dispositivos. Prossiga com cautela.

Como instalar uma GSI?

Para instalar uma GSI, seu dispositivo precisa ser compatível com o Project Treble. A partir disso:

Passo 1: Colocar o celular em modo Fastboot

Siga o tutorial em nossa página sobre Fastboot para acessar este modo.

Passo 2: Preparar a imagem da GSI

Extraia o arquivo da GSI, que geralmente vem em formatos como .zip, .tar.gz ou .gz. A imagem extraída terá a extensão .img, que será usada para instalação.

Passo 3: Limpar partições

Execute os seguintes comandos no terminal para garantir que não haverá conflitos durante a instalação:

fastboot -w fastboot delete-logical-partition product fastboot delete-logical-partition product_a fastboot delete-logical-partition product_b fastboot delete-logical-partition system_ext fastboot delete-logical-partition system_ext_a fastboot delete-logical-partition system_ext_b

Passo 4: Instalar a GSI

Use o comando abaixo para instalar a imagem da GSI:

fastboot flash system gsi.img

(Substitua gsi.img pelo nome correto da imagem extraída).

Passo 5: Reiniciar o dispositivo

Conclua o processo com o comando:

fastboot reboot

Se tudo ocorrer corretamente, o dispositivo será inicializado com a nova GSI. Aproveite!
