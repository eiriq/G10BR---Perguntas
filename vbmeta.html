O que é o VBMETA?

O vbmeta é um arquivo usado no sistema operacional Android para garantir a integridade e a segurança do boot do dispositivo. Ele é parte do Verified Boot, um recurso que verifica a integridade do software durante o processo de inicialização. O vbmeta contém uma assinatura digital que ajuda a proteger o boot loader e outros componentes críticos do sistema contra modificações não autorizadas.

Em algumas ROMs, ou GSIs (principalmente), a desativação do vbmeta é necessária para o boot da imagem (sistema) porque o Verified Boot (Inicialização Verificada) do Android pode impedir que essas imagens sejam instaladas corretamente. Neste caso, o seguinte comando deve ser usado via fastboot, para a desativação correta do vbmeta: 

fastboot flash --disable-verity --disable-verification vbmeta vbmeta.img

Certifique-se de ter o arquivo vbmeta.img no diretório onde está executando os comandos via fastboot.
